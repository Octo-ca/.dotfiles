.dotfiles
=========

![screenshot](screenshot.png)

## System requirements:

* OS X 10.10+
* Xcode 6.0.1+
* Command Line Tools for Xcode

## installation

```sh
git clone https://github.com/stoeffel/.dotfiles ~/.dotfiles
cd .dotfiles
./install
```
